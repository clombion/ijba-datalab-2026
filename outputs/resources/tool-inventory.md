## NOTES ON TOOL SELECTION

**Key patterns observed across the corpus:**

1. **Multi-model is standard.** No organization relies on a single LLM. Alviani at Ippen evaluated 200+ models. Der Spiegel uses ChatGPT, Google, and Mistral via their internal platform. JP/Politikens (Magna) and Delfi mix Azure-hosted and Hugging Face open-source models. The Economist combines DeepL with multiple LLMs for translation.

2. **Open-source for core editorial functions.** iTromsoe (Kotemon), Radio Canada (Hugging Face advocacy), Ippen Digital (LoRA fine-tuning, WhisperX), and Daudens (Essential AI Toolkit) all prioritize open-source for transparency, vendor independence, and linguistic sovereignty. DeepSeek R1 and Llama models increasingly competitive.

3. **Commercial tools for generic functions.** ChatGPT/Copilot for general prompting; specialized tools built in-house for editorial-core applications. BBC deploys Copilot organization-wide while building bespoke Style Assist internally.

4. **Local processing for sensitive work.** Whisper/WhisperX and Ollama for local inference when handling confidential sources or sensitive data. iTromsoe and Ippen Digital both emphasize local-first for investigative material.

5. **RAG as default architecture.** Every production system touching published content uses retrieval-augmented generation to ground outputs in authoritative sources. Election chatbots (Aftonbladet, SZ), archive tools (Dewey, Delfi, Economist), investigative assistants (Kotemon, LAILA), and safety tools (Jess) all rely on RAG.

6. **CMS integration drives adoption.** SZ, Aftonbladet, Ippen Digital, and JP/Politikens all embed AI directly into CMS workflows. Siegele at The Economist notes that two-click Google Docs extensions reduce friction. Tools that require leaving the workflow see lower adoption.

7. **Internal platforms replace individual licenses.** Delfi, Der Spiegel, SZ (Langdog), JP/Politikens (Magna), and Ippen Digital (AI Studio) all built internal platforms rather than buying individual ChatGPT licenses for hundreds of staff. Provides data security, cost control, and usage analytics.

8. **Agentic architectures emerging.** Ippen Digital's multi-agent virtual video teams, Delfi's browser-based data scraping agents, iTromsoe's GIN alert ecosystem, VG's agent-to-agent error checking, and Roy's Claude Code sub-agent workflows all point toward autonomous AI systems as the next phase beyond chatbot-style tools.

9. **Vibe coding democratizes tool creation.** Semafor (Chua) builds AI tools via Google App Scripts and Claude. Aftonbladet journalist Elin built a proofreading GPT with no programming experience. Ippen Digital's CTO does front-end engineering via AI coding assistants. The barrier between "journalist" and "developer" continues to collapse.

10. **Translation as killer app.** The Economist (Siegele) calls it "the most successful AI application." BBC's most mature AI project is translation across 42 languages. Delfi's cross-Baltic feed operates at near-zero cost. Good Tape (Zetland) reached $3M revenue from transcription alone.
