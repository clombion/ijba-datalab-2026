## Methodological Notes

1. **Mutual exclusivity:** Each code assigned to one theme only. Where dual relevance exists, the code is placed where it carries the most analytical weight.

2. **Collective exhaustiveness:** All 1,443 codes covered by the 12 themes. Definitions use IN/OUT boundary markers.

3. **Analytical naming:** Theme names capture core tensions, paradoxes, or insights rather than descriptive categories.

4. **Episode estimates** based on topical breadth of assigned codes, not precise episode-level tracking.

5. **Valence distribution:** Codes span positive, negative, neutral, and mixed valences, reflecting substantively ambivalent discourse. Themes organized by domain, not valence.

6. **Representative quotes placeholder:** The JSON codebook includes a placeholder flag indicating that illustrative quotes from episode transcripts should be added in a subsequent pass.
